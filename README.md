Cloning the Project:

1.  Open your terminal or command prompt.
2.  Navigate to the directory where you want to store the project.
3.  Clone the repository using the following command:

    ```bash
    git clone https://github.com/Alyasa62/Hunrmand
    ```

4.  Open the cloned folder in Android Studio.
5.  Let Android Studio build the project and sync the Gradle files.
